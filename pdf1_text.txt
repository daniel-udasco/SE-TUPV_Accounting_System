3.  Proposed  System  Features  
3.1  Core  Features:  
-  Online  Transactions  -  UX  friendly  forms  for  transactions  -  Reliable  account  verification  using  student  ID  information,  for   -  Display  information  like  the  location  of  the  office,  Board  of  Staffs,  etc.  
 
3.2  Digital  Financial  Services:  
-  Applying  for  a  summer  class  -  Paying  for  school  fees  -  School  materials  Shop  (uniform  textile,  lanyard,  and  ID)  
 
3.3  User  Roles  &  Permissions:  
-  Admin  and  Staff:  Manage  Transactions,  Verify  Students,  and  everything  related  to  
accounting
 
and
 
finance.
 -  Student:  Access  to  the  online  accounting  system  for  transactions,  etc.
